Brassica Gene Expression Database

Period: 2024.04 - 2025.02

Type: National Undergraduate Innovation Project

Project Overview

A comprehensive database platform for analyzing gene expression patterns in Brassica species. The platform integrates multiple data sources and provides various analysis tools.

Key Features

  • Integration of multiple gene expression datasets
  • Interactive visualization tools
  • Advanced search and filtering capabilities
  • RESTful API for programmatic access

Technologies Used

  • Vue.js
  • Node.js
  • MongoDB
  • D3.js
  • Docker
